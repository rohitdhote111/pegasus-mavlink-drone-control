# pegasus-mavlink-drone-control
control drone in isaac sim pegasus simulator using keyboards
Now the controls are:
space - Arm/disarm
o - Toggle offboard mode
t - Takeoff
l - Land
w - Move forward
s - Move backward
a - Move left
d - Move right
q - Rotate left
e - Rotate right
r - Move up
f - Move down
x - Quit
